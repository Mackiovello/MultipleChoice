# MultipleChoice

## Running

1. Install [NPM](https://www.npmjs.com/get-npm)
2. Run `npm install`
3. Run `npm start`
4. The app will open at localhost:3000
5. Click "start" and answer the questions

## Testing

1. Install [NPM](https://www.npmjs.com/get-npm)
2. Run `npm install`
3. Run `npm test`
4. Jest will return a test report
